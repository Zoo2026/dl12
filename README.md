# 유아용 그림책 독서 방탈출: 아기돼지 삼형제

이 프로젝트는 만 7세 유아를 대상으로 한 **'아기돼지 삼형제'** 그림책 기반 융합 교육용 웹페이지입니다. 아이들이 흥미를 느낄 수 있는 방탈출 게임 형식을 차용했습니다.

## 📁 파일 구조
- **index.html**: 웹페이지 화면 구성 (교사용 안내, 퀴즈, 미션, 보상 화면)
- **style.css**: 유아 친화적 디자인 및 애니메이션 스타일
- **script.js**: 게임의 상호작용 로직 (정답 확인, 화면 전환 등)

## 🚀 실행 방법
1. 폴더 안의 `index.html` 파일을 더블 클릭하여 웹 브라우저로 실행합니다.
2. 인터넷 연결 없이 로컬 환경에서 바로 사용 가능합니다.

## 👩‍🏫 교사 활용 가이드

### 1. 활동 전 준비
- 아이들과 함께 '아기돼지 삼형제' 그림책을 읽으셨나요? 책을 읽은 후 독후 활동으로 이 프로그램을 활용하면 더욱 효과적입니다.
- **주제**: "늑대로부터 안전한 튼튼한 집 짓기"

### 2. 단계별 활동 안내
- **도입**: 화면의 안내를 읽어주며 "늑대가 나타나기 전에 막내 돼지를 도와주자!"고 동기를 유발합니다.
- **미션 1 (내용 이해)**: 벽돌 집이 얼마나 튼튼한지 퀴즈를 풉니다.
- **미션 2 (수학)**: 화면에 흩어진 벽돌(6개)을 아이들과 함께 하나씩 세어봅니다.
- **미션 3 (언어)**: '벽돌집'이라는 단어를 입력하여 미션을 완성합니다.
- **마무리**: 늑대를 물리친 것을 축하하며 포춘쿠키 속 인성 메시지를 함께 읽고 실천을 다짐합니다.
